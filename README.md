# Tesis Machine Learning

Repositorio correspondiente a mi tesis de pregrado, enfocada en el uso de Machine Learning para predecir funciones ejecutivas y habilidades matemáticas en niños a partir de datos neuropsicológicos.

## Descripción
En este trabajo se implementaron y compararon distintos modelos supervisados, incluyendo:
- Decision Tree
- Random Forest
- XGBoost
- CatBoost
- LightGBM

Los modelos fueron evaluados mediante métricas de error y análisis estadístico para comparar su desempeño y analizar la relevancia de las variables predictoras.

## Contenido del repositorio
- `Memoria MAO.pdf`: documento de tesis
- `Presentación.ipynb`: notebook con análisis o visualizaciones de apoyo
- `PruebasMemoria.ipynb`: notebook con pruebas y experimentos de modelamiento
